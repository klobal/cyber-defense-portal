## 📋 System Overview & Vision

The **Universal Cyber Defense & Forensic Intake Portal** is a production-grade, zero-dependency, multi-page software infrastructure designed to serve as an elite digital safety desk. This platform provides a rigorous, chronological framework to handle, diagnose, contain, and legally document any digital threat brought forward by any individual—whether a domestic **Kenyan Citizen** or an international **Non-Citizen** (expatriates, tourists, or foreign investors).

### The Core Problem This System Solves
Most cybersecurity response portals fail because they are built as static, single-page incident logs or rely on heavy, internet-dependent frameworks that crash during local network drops. This system is engineered as an interconnected, multi-page suite of pure semantic HTML files that run flawlessly on low-bandwidth setups, older browser engines, or offline local servers.

It removes human forgetfulness from the intake process. It guides the system operator or the user through a strict, non-skippable chronological journey: **Value Realization ➔ Upfront Financial Transparency ➔ Identity Isolation ➔ Proactive Monitoring ➔ Incident Triage ➔ Judicial Evidence Synthesis.**

## 💰 The Mandatory Financial Transparency Directive

To prevent user exploitation and build complete trust, this platform operates under a strict **No-Skipping Financial Policy**. 

Before a citizen or non-citizen can input case details, system logs, or personal documentation, they are routed through `01_welcome_and_onboarding/transparent_pricing.html`. This page presents an unskippable, itemized ledger outlining all diagnostic tariffs, verification costs, and administrative processing fees. 

### Core Tariff Framework
*   **Zero Hidden Fees:** Every cost vector (e.g., standard data validation vs. deep forensic analysis) is calculated dynamically based on checked items before data intake occurs.
*   **Equity-Based Waivers:** Clear, programmable criteria automatically scale down or entirely waive fees for low-income citizens, vulnerable student demographics, or victims under critical extortion loops.
*   **Binding Estimates:** The final generated dossier locks the pricing token to ensure the station desk or external processors cannot inflate administrative costs post-session.
